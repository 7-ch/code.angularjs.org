Error: $rootScope:inevt
Recursive $emit/$broadcast event

{0} already $emit/$broadcast-ing on scope ({1})

Description

This error occurs when the an event is $emited or $broadcasted recursively on a scope.

For example, when an event listener fires the same event being listened to.

$scope.$on('foo', function() {
    $scope.$emit('foo');
});

Or when a parent element causes indirect recursion.

$scope.$on('foo', function() {
    $rootScope.$broadcast('foo');
});
